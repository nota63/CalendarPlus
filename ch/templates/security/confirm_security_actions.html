How This Feature Works

This feature allows administrators to confirm the deletion of a user from the organization. Below is a detailed guide on how to use this feature:

  1. Review Information: Carefully review the information displayed about the organization and the user to be deleted. Ensure that the details are correct.
  2. Select Actions: Use the dropdown menu to select the actions you want to perform. These actions may include deleting the user, revoking access, or other security measures.
  3. Perform Action: Once you have selected the appropriate actions, click the "Perform Action" button to proceed. This will submit the form and execute the selected actions.
  4. Cancel: If you decide not to proceed with the deletion, you can click the "Cancel" button to return to the previous page without making any changes.
  5. Confirmation: After performing the action, you will receive a confirmation message indicating the success or failure of the operation. Make sure to read this message carefully.

Please note that deleting a user is a permanent action and cannot be undone. Ensure that you have the necessary permissions and have double-checked all information before proceeding.
